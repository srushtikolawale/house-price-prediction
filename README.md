# 🏠 House Price Prediction

A Machine Learning project that predicts house prices using Linear Regression. The model is trained on housing data to estimate property prices based on different features.

---

## 🚀 Features

- Data Preprocessing
- Feature Selection
- Linear Regression Model
- Model Training & Prediction
- Performance Evaluation

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## ▶️ Run

```bash
pip install -r requirements.txt
python house_price_prediction.py
```
